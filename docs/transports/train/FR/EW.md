# 東西線

東西線（`East-West Line`）とは、FREESERVER [メインワールド](/world/main/)の東西を結ぶ鉄道路線です。

## 概要

全線で高架路線となっています。

[南北線](N)と比べて駅間隔は広めです。

### 路線データ

駅数：5駅

複々線区間：一部（中心町西～樫木）

地下鉄区間: なし
