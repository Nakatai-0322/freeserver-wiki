# 南北線

南北線（`Nouth-South Line`）とは、FREESERVER [メインワールド](/world/main/)の南北を結ぶ鉄道路線です。

## 概要

複線で、大部分で地下鉄となっています。

[東西線](./EW)と比べて駅間隔は狭めです。

全駅で[盲導鈴](/transports/train/guide-bell)が設置されています。

### 路線データ

駅数：4駅

複線区間：全線

地下鉄区間: 一部（岩見～中心町南の一部）
