# 留置所

## 概要

FREESERVERにおける利用規約違反者を収容するFREESERVERで唯一無二の収容施設。

## 所在地

建築スポーン西側のゴールド通りに進めば右手に見えてきます。

座標:`-265,43`

![拘置所](https://imgur.com/sBgKu7y.png)

## 収容対象者

比較的軽い違反者が収容される。（Flyハックなどは即刻でBANのため。）
先にミュート等の処罰が課せられる。

## その他

見学自由。
