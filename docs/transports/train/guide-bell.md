# 盲導鈴

盲導鈴とは、誘導が必要な方を鉄道駅などに誘導するための音声誘導装置である。

現実の盲導鈴に関しては[Wikipedia](https://ja.wikipedia.org)の[記事](https://ja.wikipedia.org/wiki/盲導鈴)を参考にされたい。

FREESERVER内では[FR](/transports/train/FR/)の一部の駅に設置されている。
